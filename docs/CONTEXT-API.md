# Resume Context API Reference

## Overview
The Resume Context provides centralized state management for the entire application.
Always use the provided wrapper functions instead of direct dispatch.

## Available Functions
- updatePersonalInfo(data) - Update personal information
- setExperience(array) - Set work experience
- setEducation(array) - Set education history
- setSkills(array) - Set skills
- setProjects(array) - Set projects
- setLanguages(array) - Set languages
- setSummary(text) - Set professional summary
- setImage(url) - Set profile photo
- setTemplate(id) - Set selected template
